# PWA Icons

Voeg hier de volgende icon-bestanden toe voordat je de app deployt:

| Bestand              | Formaat  | Gebruik                      |
|----------------------|----------|------------------------------|
| `icon-192.png`       | 192×192  | Android homescreen icon      |
| `icon-512.png`       | 512×512  | Android splash + maskable    |
| `apple-touch-icon.png` | 180×180 | iOS homescreen icon         |
| `strava-logo.svg`    | SVG      | Login-knop logo              |

## Snel icons genereren

Gebruik https://realfavicongenerator.net of https://pwabuilder.com om icons te genereren.

De themakleur is `#fc4c02` (Strava oranje) met achtergrondkleur `#0d0f18`.
